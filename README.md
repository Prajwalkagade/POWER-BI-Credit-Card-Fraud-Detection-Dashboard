# Credit Card Fraud Detection Dashboard

## Project Overview
This project presents an interactive **Credit Card Fraud Detection Dashboard** built using **Power BI**. 
The dashboard analyzes credit card transactions to identify fraudulent patterns, monitor transaction trends, and segment transactions based on risk levels.

## Dashboard Pages
The Power BI report includes **5 analytical pages**:

1. **Fraud Monitoring Dashboard** – Overview of transactions and fraud metrics  
2. **Transaction Trends Analysis** – Analysis of transactions over time  
3. **Transaction Amount Analysis** – Fraud patterns based on transaction amount  
4. **Fraud Outlier Detection** – Identification of unusual transaction behavior  
5. **Risk Segmentation** – Classification of transactions into Low, Medium, and High risk

---

## Key Metrics
- Total Transactions: **284K+**
- Fraud Transactions: **492**
- Fraud Rate: **~0.2%**
- Total Transaction Amount: **25M+**

---

## Key Insights
- Fraud transactions represent a very small portion of total transactions but have significant financial impact.
- Higher transaction amounts tend to show a slightly higher probability of fraud.
- Risk segmentation helps identify suspicious transactions for further investigation.
